# EduGenie Learning Assistant - Pre-Requisites

## Basic Knowledge Required

Before starting the EduGenie project, the team should have basic knowledge of:

- Python Programming
- REST APIs
- Artificial Intelligence
- Natural Language Processing
- FastAPI
- HTML and CSS
- Git and GitHub
- Virtual Environments

## Software Requirements

- Python 3.10 or above
- Visual Studio Code
- FastAPI
- Uvicorn
- Jinja2
- Google Gemini API Key
- Git and GitHub
